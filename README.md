# Chaos-Shuffle RNG
**Chaos-Shuffle RNG**, lojistik haritalama (logistic mapping) tabanlı kaotik sistemleri ve 64-bitlik dairesel bit rotasyonlarını kullanan deneysel bir rastgele bit üretecidir.

## Özellikler
- **Kaotik Yapı:** Başlangıç durumuna (seed) aşırı hassasiyet.
- **Bit Karıştırma:** XOR kaskadı ve dairesel kaydırma ile yüksek difüzyon.
- **Analiz Odaklı:** NIST testlerine uygun bit akışı üretimi.
